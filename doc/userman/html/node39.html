Ortographic Correction Module

This module takes into account phonetic similarities and tries to provide analysis for words that would be otherwise treated as unknown words.

For instance, if a mispelled word such as cavallo is found, this module will find out that it sounds the same than a correct word in the dictionary (caballo), thus providing analysis for that otherwise unknown word.

This module is under development, and it is not operational yet.
